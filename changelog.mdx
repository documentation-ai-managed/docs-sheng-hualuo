## Upgrade Your Integration

Follow these steps to update your Sheng Hualuo integration and benefit from the latest features.

<Steps>
  <Step title="Review Changelog" icon="book-open">
    Identify breaking changes relevant to your setup, such as updated authentication headers.
  </Step>
  <Step title="Update Dependencies" icon="package">
    <CodeGroup tabs="npm,yarn">
      ````bash
      npm install sheng-hualuo@latest
      ````

      ````bash
      yarn add sheng-hualuo@latest
      ````
    </CodeGroup>
  </Step>
  <Step title="Update API Calls" icon="code">
    Replace query param auth with header:

    ````javascript
    const response = await fetch('https://api.example.com/v1/sites/verify', {
      method: 'POST',
      headers: {
        'X-API-Key': 'YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({ url: 'https://your-site.com' })
    });
    ````
  </Step>
  <Step title="Test and Deploy" icon="rocket">
    Verify your site settings in the dashboard and monitor for issues.
  </Step>
</Steps>
