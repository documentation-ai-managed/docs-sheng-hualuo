## Prerequisites

Before you start, ensure you have:

- A registered domain name for your website
- Access to your domain's DNS settings or HTML file upload capabilities
- A modern web browser

<Callout kind="info">

Sheng Hualuo is free to use. No credit card required for signup.

</Callout>

## Create a Free Account

Follow these steps to sign up quickly.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Go to [https://dashboard.example.com](https://dashboard.example.com) and click **Sign Up**.
  </Step>
  <Step title="Enter Your Details" icon="user">
    Provide your email, create a password, and agree to the terms.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox and click the verification link.
  </Step>
  <Step title="Complete Onboarding" icon="check-circle">
    Select your role (e.g., webmaster) and dashboard preferences.
  </Step>
</Steps>

## Add and Verify Your Website

Add your site URL and choose a verification method. Sheng Hualuo supports multiple options.

<Tabs>
  <Tab title="HTML Meta Tag" icon="code">
    Add this meta tag to your site's `<head>` section:

    ```
    <meta name="shenghualuo-site-verification" content="YOUR_VERIFICATION_CODE" />
    ```

    Replace `YOUR_VERIFICATION_CODE` with the code from your dashboard. The platform checks automatically within minutes.
  </Tab>
  <Tab title="DNS TXT Record" icon="settings">
    Add a TXT record to your domain's DNS:

    ```
    shenghualuo-site-verification.example.com TXT "YOUR_VERIFICATION_CODE"
    ```

    Propagation takes 5-30 minutes. Use tools like `dig` to confirm.
  </Tab>
  <Tab title="HTML File Upload" icon="upload">
    Download the verification file from your dashboard and upload it to your site's root directory as `shenghualuo-verify.html`.

    Access it via `https://your-site.com/shenghualuo-verify.html` to confirm.
  </Tab>
</Tabs>

<Callout kind="tip">

Choose the method that best fits your hosting setup. DNS is recommended for production sites.

</Callout>

## Configure Basic Settings

Once verified, set up essential configurations.

<Steps>
  <Step title="Submit Sitemap" icon="map">
    Upload or enter your sitemap URL (e.g., `https://your-site.com/sitemap.xml`).

    <CodeGroup tabs="XML,Robots">
      ```xml
      <?xml version="1.0" encoding="UTF-8"?>
      <urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
        <url>
          <loc>https://your-site.com/</loc>
          <lastmod>2024-01-15</lastmod>
          <priority>1.0</priority>
        </url>
        <url>
          <loc>https://your-site.com/about</loc>
          <priority>0.8</priority>
        </url>
      </urlset>
      ```

      ```txt
      User-agent: *
      Allow: /
      Sitemap: https://your-site.com/sitemap.xml
      ```
    </CodeGroup>
  </Step>
  <Step title="Set Geographic Target" icon="globe">
    In site settings, select your primary country or region for optimized crawling.
  </Step>
  <Step title="Enable AI Insights" icon="zap">
    Toggle AI-driven search optimization in the advanced settings panel.
  </Step>
</Steps>

## Run Your First Performance Check

Check your site's index status and performance metrics.

Use the API for programmatic checks:

<Request tabs="cURL,JavaScript">
  ```bash
  curl -X GET "https://api.example.com/v1/sites/your-site.com/performance" \
    -H "Authorization: Bearer YOUR_API_KEY"
  ```

  ```javascript
  const response = await fetch('https://api.example.com/v1/sites/your-site.com/performance', {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  });
  const data = await response.json();
  console.log(data);
  ```
</Request>

<Response tabs="Success">
```json
{
  "site": "your-site.com",
  "indexedPages": 150,
  "performanceScore": 92,
  "aiRecommendations": ["Optimize images", "Improve mobile speed"]
}
```
</Response>
