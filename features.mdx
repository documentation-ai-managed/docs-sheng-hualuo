## Website Performance Analytics

Monitor how Bing indexes and ranks your site with detailed metrics.

<Callout kind="info">
  Access analytics via the dashboard at `https://dashboard.example.com/analytics`.
</Callout>

Use the performance dashboard to view key metrics like impressions, clicks, and average position.

<Tabs>
  <Tab title="Impressions & Clicks" icon="activity">
    Review query-level data to identify top-performing pages.

    ```javascript
    // Fetch performance data via API
    const response = await fetch('https://api.example.com/v1/performance', {
      headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
    });
    const data = await response.json();
    console.log(data.impressions); // e.g., 15000
    ```
  </Tab>
  <Tab title="Index Coverage" icon="search">
    Check crawl errors and index status.

    ```javascript
    // Query index status
    const indexData = await fetch('https://api.example.com/v1/index-coverage');
    console.log(indexData.validPages); // e.g., 2500
    ```
  </Tab>
</Tabs>

## Leveraging AI-Driven Search Insights

Sheng Hualuo uses AI to provide proactive recommendations.

<Expandable title="Advanced AI Features" default-open="false">

AI analyzes your site's content against Bing's index to suggest improvements.

```javascript
// Get AI recommendations
const aiRecs = await fetch('https://api.example.com/v1/ai-insights', {
  method: 'POST',
  body: JSON.stringify({ siteId: 'your-site-id' })
});
```

<CodeGroup tabs="JavaScript,Python">
```javascript
// Process recommendations
aiRecs.forEach(rec => {
  if (rec.priority === 'high') {
    console.log(`Optimize: ${rec.keyword}`);
  }
});
```
```python
# Process recommendations
for rec in ai_recs:
    if rec['priority'] == 'high':
        print(f"Optimize: {rec['keyword']}")
```
</CodeGroup>

</Expandable>
