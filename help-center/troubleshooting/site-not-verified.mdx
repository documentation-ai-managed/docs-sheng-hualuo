---
title: Site Not Verified
description: Resolve verification issues in Bing Webmaster Tools.
---

## Symptoms

- Site added but still unverified
- "Verification failed" message

## Common Causes

- Verification file not uploaded correctly
- DNS or meta tag changes not taking effect

## Quick Fix

<Steps>
  <Step title="Check verification method">
    Make sure the verification file, meta tag, or DNS record is live and matches what Bing provided.
  </Step>
  <Step title="Revalidate in Webmaster Tools">
    Return to the site dashboard and click "Verify" again.
  </Step>
</Steps>
